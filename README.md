# Tenet Logica Guild Website

Welcome to the Tenet Logica Guild website—a playful, arcane twist on the classic software consultancy homepage. This site is built with Angular and designed for developers who vibe like mages.

## Project Vision
Tenet Logica is a closed guild of developers (not a company), led by Gandalf and Merlin. Membership is by application only, and collaboration is informal—members share the loot and code like wizards. The site appears as a normal software consultancy, but a hidden spell reveals its true magical nature.

## Features
- **Mundane Mode:** Standard software consultancy homepage, navbar, and footer.
- **Arcane Reveal:** A magical switch (spell) transforms all content and terminology into high-fantasy mage-speak, while keeping the meaning clear for insiders.
- **No Backend:** All content is stored in constants—no API or database required.
- **Console Spell:** Footer hint: "The truth is hidden in the logs." Opening the browser console displays ASCII art of a wizard's hat and instructions to type `castSpell(<The spell that reveals Marauder's secrets>)`.
- **CSS Transition:** The spell triggers a Marauder's Map-style reveal animation, swapping all corporate buzzwords for arcane equivalents.
